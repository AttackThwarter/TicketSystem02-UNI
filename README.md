# TicketSystem - سامانه ثبت و پیگیری درخواست‌ها

## 📋 توضیحات پروژه
یک سامانه تحت وب با ASP.NET Core MVC برای ثبت و پیگیری درخواست‌ها (تیکت)

## 🛠️ تکنولوژی‌ها
- ASP.NET Core 8 MVC
- Entity Framework Core
- SQLite Database
- Bootstrap 5

## 👥 نقش‌ها
- **کاربر (User)**: ثبت درخواست و پیگیری وضعیت
- **مدیر (Admin)**: مدیریت و پاسخ به درخواست‌ها
